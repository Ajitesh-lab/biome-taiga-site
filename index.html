Browser Control System Beta

Your AI lives in your browser.

Biome ships with a background browser relay that gives the AI direct, real-time control over your Chrome browser — open tabs, navigate pages, click elements, read content, and automate anything you can do manually. No cloud infrastructure. No screen sharing. Completely local.

Browser Control → AI Chat Piggyback

Use any AI chat in your browser — no extra API key.

Because Biome already controls your browser, it can automatically relay queries to whichever AI chat you have open and pipe the response back. Works with any AI chat interface in Chrome. Your own session, your own subscription.

auto_awesome

Claude

Recommended

Biome works best with an active claude.ai session open in Chrome. Claude's longer context window and instruction-following make it ideal for complex agentic tasks via browser piggyback.

chat

Any AI chat in your browser

The piggyback system is not Claude-specific — it works with ChatGPT, Gemini, Perplexity, and any other AI chat interface you have open in Chrome. If you can type into it, Biome can use it.

* A primary model (API key or Biome plan) is still required for Biome's core reasoning and tool loop. The AI chat piggyback is an optional enhancement layer for web-augmented responses.

Security & Privacy

Built Private.
Hardened by Design.

Full Security Details arrow_forward
lock

Local First, Always

Your conversations, memory, and API keys live on your machine — never on our servers. AI calls go directly from your device to your chosen provider. We only store your email address.

verified_user

SSRF Hardened

All outbound network requests are validated against host allowlists and private-IP DNS checks. The gateway cannot be tricked into fetching from internal or unintended hosts.

gpp_maybe

Exec Approval Required

Every command the AI wants to run requires your explicit approval. Approval timeouts result in a hard block — there is no silent fallback execution path.

policy

Trust Isolation

External content — webhooks, wake events, runtime notifications — is quarantined as untrusted before entering the AI's context. Outside sources cannot inject trusted instructions.

history_toggle_off

Clean Session History

Internal tool data and system metadata are stripped from all user-facing views. What you see in chat is exactly what the model intended — no raw internals leaking through.

speed

Rate Limited Backend

Our small server footprint (account & onboarding only) enforces per-IP rate limits. OTP codes expire in 10 minutes and cannot be reused. Everything else runs entirely on your machine.
